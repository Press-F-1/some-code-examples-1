# some-code-examples-1 (Уфимцев Т. А.)
Содержит в себе примеры небольших программок. Одна - на парсинг, а другая, групповой проект, рассматриваемый с точки зрения опосредованного участия автора в написании тестов к методам классов.


## В папках Pars и Team содержатся код и зависимости

### Pars
Содержит программу для парсинга сайта с целью достать данные из конкретной таблицы + несколько автотестов функций/методов.


### Team
Групповой проект на анализ данных IMDb. Писал некоторые тесты, однако в итоговом формате от моих мало что осталось, так что это скорее для ознакомления с тем, с чем примерно приходилось работать.
